
# Interactive Love Website

A simple interactive web project built using HTML, CSS, and JavaScript.

## Features

- Clickable gift box
- Sequential image animation
- Floating heart effect
- Heart drawing using mouse
- Responsive full-screen canvas

## Technologies

- HTML
- CSS
- JavaScript

## Project Structure

```
├── index.html
├── style.css
├── script.js
```

## Note

This repository contains only the source code (HTML, CSS, and JavaScript). The image assets used in the project are not included.
